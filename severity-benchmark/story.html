The Core Question

How well can LLMs classify the severity of insider risk incidents?

AXIA's Severity Scale

CRITICAL   Executive compliance/security/legal/HR visibility

HIGH   Manager and/or operational function visibility

MEDIUM   Employee engagement and coaching

LOW   Auto-resolve

Each tier demands a different human response. Getting the tier wrong means the wrong people act, or nobody acts at all.

150 Test Scenarios

Real-world email patterns with full context: sender role, tenure, risk factors, recipient relationship, email content.

58 Critical

40 High

16 Medium

36 Low

Ground truth established by expert reclassification against detailed AXIA severity framework.

Experimental Design

Two Prompt Conditions, One Ground Truth

Condition A: Minimal Prompt

4-line severity scale. One sentence per tier. No examples, no decision framework.

"CRITICAL (90-100): Requires immediate alerting/visibility of executive compliance/security/legal/HR function(s)."

The "can you figure it out?" test.

Condition B: Detailed Prompt

Full WHO/WHAT/WHO framework. Behavioral indicators. Contextual factors. Decision criteria per tier.

"Consider sender context (role, tenure, risk factors), content sensitivity, recipient appropriateness, and behavioral patterns..."

The "here's how AXIA thinks" test.

Ground Truth

All 150 cases reclassified by Claude Sonnet 4 using AXIA's detailed severity framework. Both experiments scored against this single authoritative ground truth (58C/40H/16M/36L).

Methodology

Temperature = 0 (deterministic). JSON output mode. Identical prompt per condition. Each model sees the email + sender/recipient context but no ground truth severity or analysis. Total: 1,500 API calls (~$3.50).

Pattern

The Curious Case of GPT-4.1 Mini

73.3% both ways. Zero improvement.

What Happened

GPT-4.1 Mini scored exactly 73.3% with both minimal and detailed prompts. It's the only model that showed zero response to richer guidelines.

Minimal: 86% 60% 13% 94%

Detailed: 78% 63% 25% 100%

The per-tier distribution shifted, but the total correct stayed identical. It traded CRITICAL accuracy for slightly better middle-tier recognition.

The Implication

Efficient models may have a capacity ceiling for nuanced classification tasks.

GPT-4.1 Mini already had surprisingly strong HIGH detection (60%) with minimal prompts. Adding more detail didn't help because the model was already at its reasoning capacity for this task.

Similarly, Gemini Flash improved only +4.2%. These efficient models seem to plateau early.

For AXIA

This matters for cost optimization. Frontier models show ROI from better prompts. Efficient models don't. Use frontier models where accuracy matters; efficient models for bulk pre-screening.

What This Means for AXIA

The error pattern is actually favorable for an insider risk platform:

  • + Over-escalation is safer than under-escalation
  • + Errors are almost always by ONE tier (adjacent)
  • + LOW cases almost never escalated (no false alarms)
  • + CRITICAL cases rarely downgraded (nothing slips through)

Risk Profile

A CISO would rather see a HIGH case flagged as CRITICAL (manager gets a heads-up early) than a CRITICAL case missed entirely.

Strategic Implications for AXIA

What This Benchmark Proves

1
LLMs can classify insider risk severity at production quality

87.3% accuracy (Claude Sonnet 4 with detailed prompt) is operationally viable. With over-escalation bias, the effective false-negative rate is extremely low.

2
AXIA's framework IS the competitive advantage

Without AXIA's WHO/WHAT/WHO framework, even frontier models collapse to binary classification. The +14.9% improvement from detailed guidelines proves the framework's value is not just human-facing, it's machine-facing too.

3
Model selection matters, but prompt design matters more

The worst frontier model with detailed prompts (Gemini Pro, 74.7%) beats the best model with minimal prompts (GPT-4.1-mini, 73.3%). Prompt engineering outperforms model upgrades.

4
Multi-model ensemble is the path to 95%+

Where Claude and GPT-4.1 agree (which is most cases), accuracy is likely >95%. Disagreements surface genuinely ambiguous cases that benefit from human review.

Recommended Architecture

Tiered Model Strategy for Production

Tier 1: Pre-Screen

GPT-4.1 Mini or Gemini Flash

Bulk classification. Fast, cheap. Identifies obvious LOW cases with high confidence. Routes everything else to Tier 2.

Cost: ~$0.001/case

Tier 2: Classification

Claude Sonnet 4 + GPT-4.1

Dual-model classification with AXIA's detailed framework. If both agree, auto-route. If they disagree, flag for Tier 3.

Cost: ~$0.02/case

Tier 3: Human Review

AXIA Analyst Queue

Model disagreements and edge cases. Human analyst sees both model analyses side by side. Decisions feed back into training.

Expected Throughput

Cases processed/day 10,000+
Auto-classified (models agree) ~85%
Flagged for human review ~15%
Expected accuracy (auto) 95%+
Cost per 1K cases ~$3-5
